使用CMD/Powershell映射网络驱动器

2022-03-14

Win11 Build 22567 映射网络驱动器会导致无限绿屏，升级到 22572 不绿屏了可是映射网络驱动器的选项没了？？微软可真有你的

映射网络驱动器

打开 Powershell 或者 CMD 执行：

net use Z: \\192.168.123.1\Data1 Password /user:Username

Z是网络驱动器的盘符。
192.168.123.1\Data1 是网络驱动器地址。
Password 是连接密码。
Username 是用户名。
把这些参数按照自己的替换掉就行。

删除网络驱动器

删除就比较简单了
打开 Powershell 或者 CMD 执行：

net use Z: /del

Z 是网络驱动器的盘符。
