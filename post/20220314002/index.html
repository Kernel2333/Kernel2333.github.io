资源管理器里不显示网络驱动驱动器

2022-03-14

Win 11 Build 22572 的资源管理器里不显示映射的网络驱动器...
解决方案很简单，就是映射完之后重启资源管理器，不知道为什么 22572 重启电脑无效，必须要手动重启资源管理器

先看看有没有映射成功

以管理员权限打开 Powershell 或者 CMD 执行：

net use

有映射成功的话应该能在这里看到

Taskkill 结束Explorer进程

以管理员权限打开 Powershell 或者 CMD 执行：

taskkill /F /IM explorer.exe

启动Explorer进程

以管理员权限打开 Powershell 或者 CMD 执行：

start explorer

然后再打开资源管理器应该就能看到映射的网络驱动器了
